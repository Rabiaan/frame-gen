# Frame Gen

A modern web application built with React, Vite, and Tailwind CSS.

## Features

- Responsive design
- Modern UI components
- Interactive animations
- Contact form with EmailJS integration
- Multiple pages: Home, About, Services, Projects, Contact, Privacy Policy, Terms of Service

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Chart.js
- React Router DOM
- Lucide React Icons

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

This project is configured for deployment on Vercel. To deploy:

```bash
# Install Vercel CLI
npm i -g vercel

# Deploy to Vercel
vercel --prod
```

The deployment will be created as "frame-gen" on Vercel.
